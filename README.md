## 使用教程
#### init 项目
* 安装 ```npm install @lianpf/create-app-cli -g```
* 初始化项目 ```ca init <template-name> <project-name>```

#### 其他命令(开发中)
* ```ca config get```
* ```ca config set type orgs```
* ```ca config set registry vuejs-templates```
* ```ca config set type users```
* ```ca config set registry YvetteLau```

#### 项目模板
* ```react-template```
* ```webpack-template```
